# <img src="static/img/logo.svg" alt="Phoenix Forge Lab logo" height="26" align="top" /> Phoenix Forge Lab

> Deep, no-fluff notes on the things I'm learning — just sharing what I figure out as I go.

A growing collection of deep, no-fluff notes — just me sharing what I figure out as I learn
it. Each one goes past *how to use it* down to **how it actually works underneath**, the why
behind the functionality and the design.

**Live site:** https://DevGeekPhoenix.github.io/phoenix-forge-lab/

I'm turning each topic into a [NotebookLM](https://notebooklm.google.com) notebook so you can
ask it questions and explore interactively — look for the 🧠 callout (added per topic as I
publish them).

## Topics

- **MongoDB** — documents and BSON, the WiredTiger storage engine, B-tree indexes,
  aggregation, data modeling, replication & sharding.
- **PostgreSQL** — coming next.

## Built with

[Docusaurus](https://docusaurus.io/) + [Mermaid](https://mermaid.js.org/) diagrams, deployed
to GitHub Pages.

## Run it locally

```bash
pnpm install
pnpm start        # dev server at http://localhost:3000/phoenix-forge-lab/
pnpm build        # production build into ./build
```
